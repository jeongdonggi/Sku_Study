# 소켓 통신

## 통신

다스베이더: 친구 사귀고 싶음 -> 전화기를 사서 친구에게 전화를 걸려고 함 -> 제다이가 방해함

## Connection:
- Network: 통신 창구
- Protocol: 통신 규칙(언어)
TCP:연결지향형 통신 규약(3hand-shake, 소켓 안전), UDP: 비연결지향형 통신규약(받든말든 그냥 보냄)
- Socket: 수화기
양방향 통신의 하나의 종착지(응용 계층[HTTP])
- Address: 전화번호
IP: 컴퓨터 식별용 위치 주소(IPv4, IPv6), Port: 컴퓨터 내 접속 창구 번호
- Data: 타입(Byte, String, JSON)
- Firewall: 통신 방해자(방화벽)
- Port-forwarding: 
특정 집 컴퓨터로 보낼 때 공유기로 가게 되는데 기기별로 포트가 다름 -> 공유기에 포트별로 기기르 정해주면 됨
(80은 웹 사이트 사용 포트라 사용하면 안됨)

## socket 모듈 기반 통신 구현
생성: 소켓 생성, 소켓 연결
송신: 데이터 준비,데이터 암호화, 데이터 송신
수신: 데이터 수신, 데이터 복호화, 데이터 확인
